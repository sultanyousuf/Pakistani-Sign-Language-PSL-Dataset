# Pakistani-Sign-Language-PSL-Dataset
This dataset contains hand gesture images representing Pakistani Sign Language (PSL) signs for common words and phrases. It was developed to support research in sign language recognition and to help bridge the communication gap for the deaf and hard-of-hearing Urdu community in Pakistan.
--
## 📂 Dataset Details

| Property | Details |
|---|---|
| **Total Classes** | 40+ (common words & phrases) |
| **Total Images** | 8000+ |
| **Image Format** | JPG / PNG |
| **Data Type** | Static hand gesture images |
| **Angles** | Multiple angles per sign for robustness |
| **Language** | Pakistani Sign Language (PSL) For Urdu Community |

## 🧾 Classes

The dataset covers **40+ common PSL words and phrases** including everyday communication signs. Each class folder is named with the corresponding sign meaning.

---

## 📸 Data Collection

- Images were **originally collected** by the dataset author
- Each sign was captured from **multiple angles** to improve model generalization and robustness
- Data was collected under **informed consent** from participants
- No duplicate images — each image is unique
